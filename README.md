# HMCTS Task Management App (Frontend)

This is the React frontend for the HMCTS Task Management App.  
It connects to the Node.js/Express backend to create and view tasks.

- Task creation form with:
  - Title (required)
  - Description (optional)
  - Status (Pending, In Progress, Completed)
  - Due date/time (required)
- Confirmation message displayed after task creation


- Node.js (v18+ recommended)  
- npm  
- Backend server running on `http://localhost:5000`  

---



1. Navigate to the frontend folder:

npm start
